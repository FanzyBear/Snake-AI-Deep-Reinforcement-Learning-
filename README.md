#  Snake AI with Deep Reinforcement Learning

An intelligent agent that learns to play the classic Snake game using Deep Q-Learning (DQN), a reinforcement learning algorithm that combines Q-Learning with deep neural networks.

##  Features

- **Deep Q-Network (DQN)** implementation using PyTorch
- **Experience Replay** for stable training
- **Epsilon-Greedy Policy** with decay for exploration-exploitation balance
- **Custom Neural Network** architecture for Q-value approximation
- **Real-time training visualization** with Matplotlib
- **Modular code structure** for easy experimentation

##  Technologies Used

- **Python** - Primary programming language
- **PyTorch** - Deep learning framework for neural networks
- **Pygame** - Game environment and visualization
- **Matplotlib** - Training progress visualization
- **NumPy** - Numerical computations

##  AI Techniques

- **Reinforcement Learning** - Deep Q-Learning (DQN)
- **Neural Networks** - Multi-layer perceptron for Q-function approximation
- **Experience Replay** - Breaking temporal correlations in training data
- **Bellman Equation** - Q-value updates
- **Epsilon-Decay** - Adaptive exploration strategy

##  Results

The agent successfully learns to:
- Navigate the game environment efficiently
- Avoid collisions with walls and itself
- Strategically collect food to maximize score
- Achieve consistent high performance through training
